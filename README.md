# 100 Days Before 30

A single-page HTML plan for the 100 days before turning 30.

## GitHub Pages

This site is designed to be published from the repository root using `index.html`.

## Progress Data

Activity records live in `data/progress.json`. For CS, Guqin, and English, `duration` is minutes. For Exercise activities, `duration` is converted to points with the weights in `index.html`: Running `1`, Walking `0.5`, and Ping Pong/Table Tennis `1`.
